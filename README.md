# Student Grade Tracker (Java)

This is a simple console-based Student Grade Management System built using Java.

## Features
- Add new students
- Display student records
- Generate summary report
- Calculate grades automatically
- Show topper

## Technologies Used
- Java
- ArrayList
- OOP Concepts

## How to Run
1. Compile the program:
   javac StudentGradeTracker1.java
2. Run:
   java StudentGradeTracker1
